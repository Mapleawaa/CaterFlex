# 餐饮行业灵活用工小程序平台

## Core Features

- 兼职人员档案管理

- 智能岗位匹配

- 人脸识别GPS打卡

- 用工需求发布

- 自动协议生成

- 薪资结算系统

- 合规监控预警

- 数据分析仪表板

## Tech Stack

- WeChat 小程序开发框架 
- Node.js 18+

## Design

采用Material Design风格，蓝色主色调配橙色强调色，卡片式布局设计，注重信息层次和操作便捷性，适配小程序界面特点
